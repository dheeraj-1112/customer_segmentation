# customer_segmentation
notebook description

Synthetic Data Generation — 300 customers with Age, Annual Income, Spending Score & Purchase Frequency
EDA — Distributions, correlation heatmap
Preprocessing — StandardScaler normalization
Optimal K Selection — Elbow method + Silhouette score (K=4 is best)
K-Means Model — Trained and evaluated
Cluster Profiling — Auto-labeled segments:

💎 Premium Customers
🛍️ Impulse Buyers
💰 Savers / High Earners
🏷️ Budget Shoppers


Visualizations — Scatter plots, PCA 2D view, Radar charts
CSV Export — Saves segmented results
